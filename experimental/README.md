# Experimental/Archived Modules

This directory contains scaffold modules that were moved from the main source tree because they were either:
1. Minimal implementations not actively developed
2. Early-stage prototypes awaiting further development
3. Deprecated in favor of other implementations

These modules are archived for reference and potential future use.

## Archived Modules

- `file_diff.jl` - File comparison scaffold
- `git_readonly.jl` - Git operations scaffold  
- `analyze_logs.jl` - Log analysis scaffold
- `compare_strategies.jl` - Strategy comparison scaffold
- `estimate_risk.jl` - Risk estimation scaffold
- `simulate_outcomes.jl` - Outcome simulation scaffold
- `summarize_state.jl` - State summarization scaffold
- `task_scheduler.jl` - Task scheduling scaffold

## Reactivation

To reactivate any of these modules, move them back to their original location in `adaptive-kernel/capabilities/`.
